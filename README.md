
## 💻 Running AgentDet Locally

AgentDet是一个基于openhands框架重构的Docker镜像漏洞检测Agent

> [!WARNING]
> On a public network? See original [Hardened Docker Installation Guide](https://github.com/All-Hands-AI/OpenHands)
> to secure your deployment by restricting network binding and implementing additional security measures.


```bash
docker pull docker_iamge_name

#前置条件
# nodejs>=v22
# python=3.12
# poetry
# docker
# gpt_api


make build
poetry install --with evaluation

conda deactivate

eval "$(poetry env activate)"

# 运行
poetry run ./evaluation/benchmarks/agent_dev/scripts/run_infer.sh llm.doubao HEAD CodeActAgent 1

```

When you open the application, you'll be asked to choose an LLM provider and add an API key.
你可以在config.toml中添加你的API key

```toml

model="model_name"
api_key="API_KEY"
base_url="BASE_URL"
```
